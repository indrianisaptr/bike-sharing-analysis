## Deskripsi Proyek
Proyek ini menganalisis pola penyewaan sepeda berdasarkan musim, kondisi cuaca, dan jenis hari menggunakan Bike Sharing Dataset.

## Setup Environment - Anaconda
```
conda create --name bike-ds python=3.9
conda activate bike-ds
pip install -r requirements.txt
```

## Setup Environment - Shell/Terminal
```
python -m venv venv
source venv/bin/activate      # macOS/Linux
venv\Scripts\activate         # Windows
pip install -r requirements.txt
```

## Run Notebook
Buka file notebook.ipynb dan jalankan seluruh cell secara berurutan.


## Run steamlit app
```
streamlit run dashboard/dashboard.py
```
